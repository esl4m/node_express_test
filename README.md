# node_express_test
First start with NodeJs and express framework
```
mkdir myapp
cd myapp
npm init
npm install express --save
```
(to add the ejs engine support)
```
express --view=ejs
```
install npm :
```
cd . && npm install
DEBUG=myapp:* npm start  (mac and linux)
```

Go to your browser 
```
http://localhost:3000/
```
